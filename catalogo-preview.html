Catálogo de Templates

Escolha o template certo na hora de criar conteúdo. Cada um é re-renderizável: pegue um, troque o conteúdo e gere a imagem.

node src/cli.js render <template> --var titulo="Texto" --output output/meu-post.png

Marca Cienty (on-brand)

Genéricos

_cienty/ não é template — é a pasta de assets compartilhados da marca (logo, ilustrações, tokens).  ·  Para conteúdo Cienty, siga a skill cienty-brand.  ·  Tamanhos disponíveis: node src/cli.js presets.
